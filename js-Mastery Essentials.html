Giới thiệu

JavaScript là một trong những ngôn ngữ lập trình phổ biến và mạnh mẽ nhất hiện nay, được sử dụng để xây dựng các ứng dụng web, mobile, và thậm chí cả ứng dụng desktop. Nếu HTML là bộ xương và CSS là lớp da của trang web, thì JavaScript chính là trái tim giúp trang web trở nên sống động và có khả năng tương tác.

Bài viết này sẽ hướng dẫn bạn nắm vững các khái niệm nền tảng trong JavaScript, bao gồm: biến (variables), hàm (functions), đối tượng (objects), và các tính năng mới của ES6+. Dù bạn là người mới bắt đầu hay đã có kinh nghiệm lập trình, hiểu rõ những phần này sẽ giúp bạn tiến xa hơn trong việc phát triển ứng dụng web hiện đại.

1. Biến (Variables) – Nền tảng của JavaScript

Biến trong JavaScript dùng để lưu trữ dữ liệu. Từ ES6 trở đi, có ba cách để khai báo biến:

  • var – dùng trong các phiên bản cũ, có phạm vi function (function scope).
  • let – có phạm vi khối (block scope), thường được dùng trong vòng lặp hoặc câu điều kiện.
  • const – dùng cho hằng số, giá trị không thể thay đổi sau khi gán.
var name = "Dev";
let age = 25;
const country = "Vietnam";

console.log(`Xin chào, tôi là ${name}, ${age} tuổi, đến từ ${country}.`);

Việc sử dụng let và const giúp mã của bạn an toàn hơn, tránh bị "rò rỉ phạm vi" hoặc ghi đè biến ngoài ý muốn.

2. Kiểu dữ liệu (Data Types)

JavaScript là ngôn ngữ động (dynamic typing), nghĩa là bạn không cần khai báo kiểu dữ liệu khi tạo biến. Các kiểu dữ liệu cơ bản gồm:

  • String – Chuỗi ký tự, ví dụ: "Hello"
  • Number – Số (cả nguyên và thực)
  • Boolean – Giá trị đúng hoặc sai (true/false)
  • Object – Tập hợp các cặp khóa–giá trị
  • Array – Danh sách các phần tử
  • Null và Undefined – Đại diện cho "không có giá trị"

💡 Dynamic Typing

JavaScript tự động xác định kiểu dữ liệu tại runtime. Một biến có thể chứa các kiểu dữ liệu khác nhau tại các thời điểm khác nhau.

3. Hàm (Functions) – Linh hồn của chương trình

Hàm giúp tái sử dụng mã và tổ chức logic chương trình rõ ràng hơn. Bạn có thể định nghĩa hàm bằng nhiều cách khác nhau: 

// Cách khai báo thông thường
function greet(name) {
  return `Xin chào, ${name}!`;
}

// Cách viết với arrow function (ES6)
const greetArrow = (name) => `Xin chào, ${name}!`;

console.log(greet("Developer"));
console.log(greetArrow("Coder"));

Arrow function mang lại cú pháp ngắn gọn và tự động ràng buộc ngữ cảnh this – điều rất hữu ích khi làm việc trong các đối tượng hoặc callback.

4. Đối tượng (Objects) – Cấu trúc dữ liệu mạnh mẽ

Mọi thứ trong JavaScript đều là đối tượng (object), hoặc có thể được coi như một đối tượng. Đối tượng cho phép lưu trữ và quản lý thông tin dưới dạng cặp key: value. 

const student = {
  name: "Dev Student",
  age: 25,
  course: "Software Engineering",
  greet() {
    console.log(`Xin chào, tôi là ${this.name} và tôi học ${this.course}.`);
  }
};

student.greet();

Bạn có thể thêm, xóa hoặc chỉnh sửa thuộc tính của object một cách linh hoạt, và kết hợp chúng với hàm để mô phỏng các đối tượng trong thế giới thực (OOP – Lập trình hướng đối tượng).

5. Mảng (Arrays) và các phương thức quan trọng

Mảng là danh sách có thứ tự, dùng để lưu trữ nhiều giá trị trong một biến. JavaScript cung cấp nhiều phương thức mạnh mẽ để xử lý mảng, chẳng hạn như map(), filter(), và reduce(). 

const scores = [80, 92, 75, 60, 100];

const passed = scores.filter(score => score >= 80);
console.log("Điểm đạt:", passed);

const average = scores.reduce((sum, score) => sum + score, 0) / scores.length;
console.log("Điểm trung bình:", average);

📌 Array Methods Cheatsheet

  • map() - Biến đổi từng phần tử
  • filter() - Lọc phần tử theo điều kiện
  • reduce() - Gộp các phần tử thành một giá trị
  • forEach() - Thực thi hàm cho từng phần tử
  • find() - Tìm phần tử đầu tiên thỏa điều kiện

6. ES6+ Features – Những cải tiến hiện đại

Phiên bản ECMAScript 6 (ES6) đã mang đến nhiều thay đổi lớn, giúp JavaScript trở nên mạnh mẽ và dễ viết hơn. Dưới đây là một số tính năng nổi bật:

  • Template literals – cho phép nối chuỗi dễ dàng với cú pháp ${}
  • Destructuring – trích xuất dữ liệu từ mảng hoặc đối tượng
  • Spread / Rest Operator (...) – sao chép hoặc gộp dữ liệu linh hoạt
  • Modules – chia nhỏ chương trình thành các file độc lập
const person = { name: "Developer", age: 25 };
const { name, age } = person;

console.log(`Tên: ${name}, Tuổi: ${age}`);

const nums = [1, 2, 3];
const extended = [...nums, 4, 5];
console.log(extended);

7. Tổng kết

Việc hiểu sâu các khái niệm cơ bản như biến, hàm, đối tượng và tính năng ES6+ là nền tảng giúp bạn làm chủ JavaScript. Đây là bước khởi đầu cần thiết trước khi bạn tiến đến các chủ đề nâng cao hơn như bất đồng bộ (async/await), DOM manipulation, hoặc framework hiện đại như React, Vue hay Angular.
