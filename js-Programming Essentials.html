JavaScript Programming Essentials

By Thanh Hiếu October 19, 2024 Thời gian đọc: 15 phút

Vì sao nên học JavaScript?

JavaScript là "ngôn ngữ của web" – thứ khiến các trang web trở nên sống động, biến những đoạn HTML tĩnh và CSS đơn giản thành những trải nghiệm tương tác.

Với JavaScript, bạn có thể làm được cả front-end (giao diện người dùng) lẫn back-end (máy chủ qua Node.js), tạo nên những ứng dụng thực sự hoàn chỉnh.

Back-End

Các khái niệm cốt lõi

Biến (Variables)

Biến là "hộp chứa giá trị". JavaScript cung cấp ba cách khai báo:

const PI = 3.14159;  // Không thể thay đổi
let count = 0;       // Có thể thay đổi
var oldWay = "Avoid"; // Cách cũ

So sánh var, let, const

Keyword Scope Hoisting Reassignable
var Function scope Yes (undefined) Yes
let Block scope Yes (TDZ) Yes
const Block scope Yes (TDZ) No

Kiểu dữ liệu

JavaScript có hai nhóm kiểu dữ liệu chính:

// Primitive types
const age = 25;           // Number
const name = "John";      // String
const isStudent = true;   // Boolean
const nothing = null;     // Null
let undefinedVar;         // Undefined
const unique = Symbol();  // Symbol

// Reference types
const person = {          // Object
  name: "John",
  age: 25
};
const colors = ["red", "green", "blue"];  // Array
const greet = function() {                // Function
  console.log("Hello!");
};

Type Coercion

JavaScript tự động chuyển đổi kiểu dữ liệu trong các phép toán:

console.log("5" + 3);      // "53" (string concatenation)
console.log("5" - 3);      // 2 (numeric subtraction)
console.log(true + 1);     // 2 (true becomes 1)
console.log(null == undefined); // true

JS là ngôn ngữ "dynamically typed" — một biến có thể thay đổi kiểu giá trị trong lúc chạy.

Hàm (Functions)

Hàm giúp gom nhóm các đoạn mã có cùng mục đích:

1. Function Declaration

function calculateArea(width, height) {
  return width * height;
}

2. Function Expression

const calculateArea = function(width, height) {
  return width * height;
};

3. Arrow Function

const calculateArea = (width, height) => {
  return width * height;
};

// Ngắn gọn hơn với một biểu thức
const calculateArea = (width, height) => width * height;

4. Higher-Order Functions

Hàm có thể nhận hàm khác làm tham số hoặc trả về hàm:

// Hàm nhận hàm khác làm tham số
function operate(a, b, operation) {
  return operation(a, b);
}

const result = operate(5, 3, (x, y) => x * y);
console.log(result); // 15

Vòng lặp và điều kiện

If-Else

const age = 18;

if (age >= 18) {
  console.log("Đã đủ tuổi");
} else if (age >= 16) {
  console.log("Gần đủ tuổi");
} else {
  console.log("Chưa đủ tuổi");
}

Switch-Case

const day = "Monday";

switch (day) {
  case "Monday":
    console.log("Start of week");
    break;
  case "Friday":
    console.log("Weekend is coming");
    break;
  default:
    console.log("Regular day");
}

Vòng lặp

// For loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While loop
let count = 0;
while (count < 5) {
  console.log(count);
  count++;
}

// For...of (cho arrays)
const fruits = ["apple", "banana", "orange"];
for (const fruit of fruits) {
  console.log(fruit);
}

DOM & Sự kiện

DOM giúp JavaScript tương tác với trang web:

Truy xuất phần tử

// Theo ID
const header = document.getElementById('header');

// Theo class (phần tử đầu tiên)
const btn = document.querySelector('.btn-primary');

// Theo class (tất cả phần tử)
const buttons = document.querySelectorAll('.btn');

// Theo tag name
const paragraphs = document.getElementsByTagName('p');

Thao tác với DOM

// Thay đổi nội dung
element.textContent = "New text";
element.innerHTML = "Bold text";

// Thay đổi style
element.style.color = "red";
element.style.fontSize = "20px";

// Thêm/xóa class
element.classList.add('active');
element.classList.remove('inactive');
element.classList.toggle('hidden');

// Tạo phần tử mới
const newDiv = document.createElement('div');
newDiv.textContent = "I'm new!";
parentElement.appendChild(newDiv);

Sự kiện (Events)

const btn = document.querySelector('#myBtn');

// Thêm event listener
btn.addEventListener('click', function(event) {
  console.log('Button clicked!');
  console.log('Event type:', event.type);
});

// Các sự kiện phổ biến
element.addEventListener('click', handleClick);
element.addEventListener('mouseover', handleHover);
element.addEventListener('keydown', handleKeyPress);
element.addEventListener('submit', handleFormSubmit);

ES6+ Features

ES6 mang lại nhiều cải tiến quan trọng:

Template Literals

const name = 'Hiếu';
const age = 25;

// Thay vì: "Xin chào " + name + ", bạn " + age + " tuổi"
console.log(`Xin chào ${name}, bạn ${age} tuổi`);

// Multi-line strings
const message = `
  Đây là
  nhiều dòng
  văn bản
`;

Destructuring

// Array destructuring
const numbers = [1, 2, 3];
const [first, second] = numbers;

// Object destructuring
const user = { name: 'Linh', age: 23, email: 'linh@example.com' };
const { name, age } = user;

// Destructuring với alias
const { name: userName, age: userAge } = user;

Spread và Rest Operators

// Spread operator
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

const obj1 = { a: 1, b: 2 };
const obj2 = { ...obj1, c: 3 }; // { a: 1, b: 2, c: 3 }

// Rest parameter
function sum(...numbers) {
  return numbers.reduce((total, num) => total + num, 0);
}
console.log(sum(1, 2, 3, 4)); // 10

Modules

// math.js
export const PI = 3.14159;
export function circleArea(radius) {
  return PI * radius * radius;
}

// main.js
import { PI, circleArea } from './math.js';
console.log(circleArea(5));

Async/Await

async function fetchUserData(userId) {
  try {
    const response = await fetch(`/api/users/${userId}`);
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }
    const userData = await response.json();
    return userData;
  } catch (error) {
    console.error('Error fetching user data:', error);
    throw error;
  }
}

// Sử dụng
fetchUserData(123)
  .then(user => console.log(user))
  .catch(error => console.error(error));

Xử lý lỗi (Error Handling)

try {
  // Code có thể gây lỗi
  const data = JSON.parse(invalidJson);
} catch (error) {
  console.error('Lỗi xảy ra:', error.message);
} finally {
  console.log('Luôn được thực thi');
}

// Custom errors
class ValidationError extends Error {
  constructor(message) {
    super(message);
    this.name = 'ValidationError';
  }
}

function validateEmail(email) {
  if (!email.includes('@')) {
    throw new ValidationError('Email không hợp lệ');
  }
}
