Giới thiệu ES6+

JavaScript là ngôn ngữ không ngừng phát triển. Với phiên bản ES6 (ECMAScript 2015) và các bản nâng cấp sau đó (ES7, ES8, …), lập trình viên được cung cấp nhiều tính năng mạnh mẽ, giúp viết code ngắn gọn, dễ đọc và hiệu suất cao hơn.

📌 Tại sao ES6+ quan trọng?

  • Code ngắn gọn hơn 30-50% so với ES5
  • Dễ bảo trì và debug hơn
  • Tương thích tốt với các framework hiện đại (React, Vue, Angular)
  • Hỗ trợ lập trình hàm và hướng đối tượng tốt hơn

Arrow Functions

Arrow Function (hàm mũi tên) là một trong những tính năng được yêu thích nhất của ES6. Nó giúp viết hàm ngắn gọn hơn và xử lý ngữ cảnh this thông minh hơn.

ES5 - Function thông thường
// Hàm thường
function sayHello(name) {
  return "Hello, " + name;
}

// Callback với function
numbers.map(function(num) {
  return num * 2;
});

// Vấn đề với this
var obj = {
  value: 42,
  getValue: function() {
    var self = this;
    setTimeout(function() {
      console.log(self.value); // Phải dùng self
    }, 100);
  }
};
ES6+ - Arrow Function
// Arrow function
const sayHello = (name) => `Hello, ${name}`;

// Callback ngắn gọn
numbers.map(num => num * 2);

// Không cần bind this
const obj = {
  value: 42,
  getValue: function() {
    setTimeout(() => {
      console.log(this.value); // this được giữ nguyên
    }, 100);
  }
};

💡 Khi nào nên dùng Arrow Function?

  • Nên dùng: Callback functions, map/filter/reduce, methods không cần this
  • Không nên dùng: Object methods (cần this), constructor functions, event handlers (cần this)

Destructuring Assignment

Destructuring cho phép bạn trích xuất giá trị từ mảng hoặc đối tượng nhanh chóng và gán vào biến chỉ bằng một dòng. 

// Destructuring mảng
const colors = ["red", "green", "blue"];
const [first, second, third] = colors;
console.log(first, second); // red green

// Bỏ qua phần tử
const [primary, , tertiary] = colors;
console.log(primary, tertiary); // red blue

// Destructuring object
const user = { 
  name: "User", 
  age: 25, 
  country: "VN",
  address: {
    city: "Hanoi",
    district: "Cau Giay"
  }
};

const { name, age, address: { city } } = user;
console.log(name, age, city); // User 25 Hanoi

// Destructuring với tham số hàm
function displayUser({ name, age }) {
  console.log(`${name} is ${age} years old`);
}

displayUser(user); // User is 25 years old

"Destructuring giúp code JavaScript trông như ngôn ngữ hiện đại — ngắn gọn và trực quan."

Template Literals

Template Literals (chuỗi mẫu) cho phép bạn nối chuỗi dễ dàng hơn, hỗ trợ biến và xuống dòng trực tiếp trong chuỗi. 

const name = "User";
const age = 25;
const scores = [8, 9, 10];

// Chuỗi đa dòng
const message = `Xin chào ${name}!
Bạn ${age} tuổi.
Điểm số của bạn: ${scores.join(', ')}`;

console.log(message);

// Biểu thức trong template
const calculation = `Tổng điểm: ${scores.reduce((a, b) => a + b, 0)}`;
console.log(calculation); // Tổng điểm: 27

// Template tags (nâng cao)
function highlight(strings, ...values) {
  return strings.reduce((result, str, i) => 
    `${result}${str}${values[i] || ''}`, '');
}

const highlighted = highlight`Hello ${name}, you are ${age} years old`;
console.log(highlighted);

Modules – Tổ chức mã nguồn chuyên nghiệp

Trước ES6, JavaScript không hỗ trợ import/export chuẩn. Với ES6 Modules, bạn có thể chia code thành nhiều file, tái sử dụng dễ dàng.

ES5 - Module Pattern
// math.js
var MathUtils = (function() {
  function add(a, b) { return a + b; }
  function multiply(a, b) { return a * b; }
  
  return {
    add: add,
    multiply: multiply
  };
})();

// app.js
console.log(MathUtils.add(2, 3));
ES6+ - Modules
// 📁 math.js
export const add = (a, b) => a + b;
export const multiply = (a, b) => a * b;
export default function calculator() { /* ... */ }

// 📁 app.js
import { add, multiply } from './math.js';
import calc from './math.js';

console.log(add(2, 3)); // 5
console.log(multiply(4, 5)); // 20

📦 Các cách export/import

  • Named exports: export const name = value
  • Default export: export default function
  • Import all: import * as Module from './module'
  • Dynamic import: import('./module').then(module => ...)

Các tính năng ES6+ khác nên biết

Tính năng Mô tả Ví dụ
Let & Const Khai báo biến với phạm vi block let x = 1; const PI = 3.14;
Spread Operator Mở rộng mảng/object const newArr = [...oldArr, newItem]
Rest Parameters Thu thập tham số thành mảng function sum(...numbers) { }
Default Parameters Tham số mặc định cho hàm function greet(name = 'Guest') { }
Enhanced Object Literals Khai báo object ngắn gọn const obj = { x, y, method() { } }
Promises & Async/Await Xử lý bất đồng bộ async function fetchData() { }

Ví dụ tổng hợp thực tế

Dưới đây là ví dụ thực tế kết hợp nhiều tính năng ES6+ trong một ứng dụng nhỏ:

// 📁 utils/helpers.js
export const formatCurrency = (amount, currency = 'VND') => 
  `${amount.toLocaleString()} ${currency}`;

export const debounce = (func, delay) => {
  let timeoutId;
  return (...args) => {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => func.apply(null, args), delay);
  };
};

// 📁 services/api.js
const API_BASE = 'https://api.example.com';

export const fetchUserData = async (userId) => {
  try {
    const response = await fetch(`${API_BASE}/users/${userId}`);
    const { data: user } = await response.json();
    return user;
  } catch (error) {
    console.error('Fetch failed:', error);
    throw error;
  }
};

// 📁 app.js
import { formatCurrency, debounce } from './utils/helpers.js';
import { fetchUserData } from './services/api.js';

// Sử dụng các tính năng ES6+
class UserProfile {
  constructor(userId) {
    this.userId = userId;
    this.user = null;
    this.search = debounce(this.handleSearch.bind(this), 300);
  }

  async loadUser() {
    try {
      const user = await fetchUserData(this.userId);
      const { name, age, balance, preferences: { theme = 'light' } } = user;
      
      this.user = { name, age, balance, theme };
      this.render();
    } catch (error) {
      this.showError(error.message);
    }
  }

  handleSearch(query) {
    const results = this.user.friends?.filter(friend => 
      friend.name.toLowerCase().includes(query.toLowerCase())
    ) ?? [];
    this.renderResults(results);
  }

  render() {
    const { name, age, balance } = this.user;
    const profileHTML = `
      

        
${name}

        
Age: ${age}

        
Balance: ${formatCurrency(balance)}

      

    `;
    document.getElementById('app').innerHTML = profileHTML;
  }
}

// Khởi tạo ứng dụng
const app = new UserProfile(123);
app.loadUser();
