Giới thiệu

JavaScript là ngôn ngữ không ngừng phát triển. Với phiên bản ES6 (ECMAScript 2015) và các bản nâng cấp sau đó (ES7, ES8, …), lập trình viên được cung cấp nhiều tính năng mạnh mẽ, giúp viết code ngắn gọn, dễ đọc và hiệu suất cao hơn.

Trong bài viết này, chúng ta sẽ khám phá bốn tính năng nổi bật nhất của ES6+ gồm: Arrow Functions, Destructuring, Modules và Template Literals. Mỗi phần đều có ví dụ minh họa thực tế để bạn dễ hiểu và áp dụng ngay vào dự án của mình.

1. Arrow Functions

Arrow Function (hàm mũi tên) giúp viết hàm ngắn gọn hơn và xử lý ngữ cảnh this thông minh hơn. 

// Hàm thường
function sayHello(name) {
  return "Hello, " + name;
}

// Arrow function
const sayHelloArrow = (name) => `Hello, ${name}`;

console.log(sayHelloArrow("User")); // Hello, User

✅ Ưu điểm:

  • Ngắn gọn, rõ ràng — đặc biệt khi dùng callback.
  • Không tạo ngữ cảnh this mới → phù hợp cho các sự kiện hoặc Promise.

⚠️ Lưu ý: Arrow function không nên dùng làm constructor hoặc với this động.

2. Destructuring Assignment

Destructuring cho phép bạn trích xuất giá trị từ mảng hoặc đối tượng nhanh chóng và gán vào biến chỉ bằng một dòng. 

// Destructuring mảng
const colors = ["red", "green", "blue"];
const [first, second] = colors;
console.log(first, second); // red green

// Destructuring object
const user = { name: "User", age: 25, country: "VN" };
const { name, age } = user;
console.log(name, age); // User 25

👉 Tính năng này giúp giảm code lặp, dễ đọc và cực hữu ích khi làm việc với JSON hoặc dữ liệu API phức tạp.

“Destructuring giúp code JavaScript trông như ngôn ngữ hiện đại — ngắn gọn và trực quan.”

3. Template Literals

Template Literals (chuỗi mẫu) cho phép bạn nối chuỗi dễ dàng hơn, hỗ trợ biến và xuống dòng trực tiếp trong chuỗi. 

const name = "User";
const message = `Xin chào ${name}!
Chúc bạn một ngày tốt lành.`;

console.log(message);
/*
Kết quả:
Xin chào User!
Chúc bạn một ngày tốt lành.
*/

🔹 Dấu backtick (`) thay cho dấu nháy đơn/đôi. 🔹 Biến được nhúng bằng cú pháp ${variable}. 🔹 Hữu ích khi tạo template HTML hoặc log động.

Hình ảnh minh họa cú pháp Template Literals

Hình minh họa cho thấy sự khác biệt giữa cách cũ (dùng \n để xuống dòng) và cách mới trong ES6 (dùng backtick — gọn hơn và dễ đọc hơn).

4. Modules – Tổ chức mã nguồn chuyên nghiệp

Trước ES6, JavaScript không hỗ trợ import/export chuẩn. Với ES6 Modules, bạn có thể chia code thành nhiều file, tái sử dụng dễ dàng. 

// 📁 math.js
export const add = (a, b) => a + b;
export const multiply = (a, b) => a * b;

// 📁 app.js
import { add, multiply } from './math.js';

console.log(add(2, 3)); // 5
console.log(multiply(4, 5)); // 20

✅ Ưu điểm của Modules:

  • Dễ bảo trì, tái sử dụng và kiểm thử.
  • Tránh xung đột biến toàn cục.
  • Tích hợp hoàn hảo với công cụ build hiện đại (Webpack, Vite, Rollup,...).

“Modules giúp JavaScript tiến gần hơn đến chuẩn lập trình hiện đại — có cấu trúc, có tổ chức và dễ mở rộng.”

5. Các tính năng ES6+ khác nên biết

  • Let / Const: Cải thiện quản lý biến, tránh lỗi phạm vi.
  • Spread & Rest Operator: Dễ dàng gộp hoặc tách dữ liệu.
  • Promises / Async–Await: Xử lý bất đồng bộ dễ hiểu hơn.
  • Default Parameters: Gán giá trị mặc định cho hàm.

6. Ví dụ tổng hợp

Dưới đây là ví dụ kết hợp các tính năng ES6 trong một đoạn code ngắn:

const greet = (name = "Guest") => `Hello, ${name}!`;

const user = { name: "User", age: 25 };
const { name, age } = user;

const scores = [8, 9, 10];
const total = scores.reduce((a, b) => a + b, 0);

console.log(`${greet(name)} You are ${age} years old. Total score: ${total}`);

Tổng quan các tính năng ES6+

Hình minh họa bên dưới cho thấy mối quan hệ nội bộ giữa Function, Generator, và Iterator trong ES6 — nền tảng cho nhiều tính năng nâng cao như generator functions và async/await trong JavaScript hiện đại.

7. Tổng kết

Các tính năng ES6+ đã thay đổi cách chúng ta viết JavaScript — từ những cú pháp nhỏ như arrow function đến cấu trúc module hóa hoàn chỉnh.

Nắm vững các tính năng này sẽ giúp bạn viết code:

  • Ngắn gọn và dễ bảo trì hơn.
  • Dễ làm việc nhóm và tái sử dụng.
  • Tương thích tốt với framework hiện đại (React, Vue, Angular,...).

👉 Hãy luyện tập bằng cách viết lại các đoạn code cũ theo cú pháp ES6, bạn sẽ cảm nhận rõ sự khác biệt!

“ES6+ không chỉ là bản nâng cấp – nó là nền tảng cho toàn bộ hệ sinh thái JavaScript hiện đại.”
